# URL Shortening

### Built With
* Java 8
* SpringBoot 2.3.4
* Swagger
* MySQL 5.7


### Installation

1. Clone the repo
```sh
git clone https://github.com/jaydipl/url-shorten.git
```
3. Create database with name as `url-shortening`
```sh
CREATE DATABASE url-shortening;
```
4. Start Service
```sh
mvn spring-boot:run
```

### Access Swagger URL
`http://localhost:8300/swagger-ui.html`
